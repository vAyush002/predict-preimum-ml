
Insurance Premium Prediction API 🚀

A Machine Learning powered Insurance Premium Prediction system built using FastAPI and Streamlit.

This project predicts the insurance premium category of a user based on health, lifestyle, occupation, and city-related information using a trained ML model.

📌 Features
# FastAPI backend for ML inference
# Streamlit frontend UI
# Pydantic validation
# Computed fields using @computed_field
# BMI calculation
# Lifestyle risk analysis
# City tier classification
# JSON API responses
# Machine Learning model integration using Pickle

🛠️ Tech Stack
- Python
- FastAPI
- Streamlit
- Scikit-learn
- Pandas
- Pydantic
- Uvicorn
